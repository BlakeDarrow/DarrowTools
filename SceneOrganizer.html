Installation

  1. Download the latest release from github.

  2. Under addon preferences in Blender, click ‘install’ and the select the recently downloaded zip file.

  3. Enable the addon

Warning

Sometimes you might get an error when installing saying there isn’t a module named “SceneOrganizer”, restart Blender and try enabling the addon again.
